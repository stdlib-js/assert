{{alias}}( value )
    Tests whether a value is a property key.

    Parameters
    ----------
    value: any
        Value to test.

    Returns
    -------
    out: boolean
        Boolean indicating whether value is a property key.

    Examples
    --------
    > var out = {{alias}}( 'foo' )
    true

    > var out = {{alias}}( 1 )
    true

    > var out = {{alias}}( true )
    false
